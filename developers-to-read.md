# Developers to read

http://github.com/fast-flow/react

### Development

```shell
npm run doc
# or npm run doc:debug
npm run js
# npm run js:debug
```

### Test

```shell
npm run test
```

### Publish gh-pages
```shell
npm run gh
npm run s
npm run gh-push
```

### Publish npm

```shell
npm run npm
cd output && npm publish
```

### Compound Command

npm run test-ci && npm run gh && npm run gh-push && npm run npm && cd output && npm publish
